## Anforderungen

- Raspberry Pi 5
- 10-Zoll Touchscreen
- Raspberry Pi OS
- Python 3.7+
- PyQt5
- GPIO-Tasten (optional)

## Fehlersuche

- Prüfen Sie die Logs im `logs/` Verzeichnis
- Stellen Sie sicher, dass alle Berechtigungen korrekt gesetzt sind
- Überprüfen Sie die Audio- und USB-Verbindungen
